# 2D Boss Battle Videogame 

1. Made with Typescript 
2. No additional frameworks
3. No additional libraries
4. Pure css/html
5. A little bit of AI Slop help

## Documentation
This project uses [TypeDoc](https://typedoc.org/) to generate API documentation from JSDoc/TSDoc comments in the source code.

To generate the documentation, run:
```bash
pnpm doc
```
The documentation will be generated in the `/docs` directory.
